# 3000X-gui
PyQT5 GUI for controlling and collecting data from MSO/DSO X3000 series Keysight/Agilent Scopes


# Scope File Format

- ScopeMetaData

- Acquisition Timings.npy


- Analog_Data.npy
  - Time Channel
    - Acquisition 1
      - Time Series Axis (X Axis)
  - Channel 1
    - Acquisition 1
      - Time Series Data (Y Axis)
    - Acquisition 2
      - Time Series Data
    - .....
      - .....
    - Channel 2
      - ....
        - ...
